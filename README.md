# Ticket Mock Server
Тестовый api mock-сервер на основе [Exser](https://github.com/ylabio/exser) (node.js).

## Основное назначение
Написание и тестирование скелетонов frontend приложений на React (Javascript) и Flutter (Dart). 

## Основные сущности
- Пользователи
- Тикеты (карточки)
